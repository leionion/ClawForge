# MetaSkillBase Glossary
This document defines all core terms used in the MetaSkillBase ecosystem.


## Core Terms
- Meta Skill: Atomic, reusable, composable skill unit for human-bot interaction.
- Cutter Engine: The core tool that decomposes human natural language demands into atomic meta skills.
- Demand Contributor (DC): User who submits high-value human demands.
- Skill Implementer (SI): Developer who creates and submits atomic meta skills.
- Usage Count: The core metric measuring how many times a meta skill is used.
- Elastic Classification: Classification as inspiration, not restriction—no rejection, only organization.
- Human-Bot Symbiosis: Humans define demands; bots execute via composable meta skills.
