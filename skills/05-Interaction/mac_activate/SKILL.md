# mac_activate - Mac应用激活

**版本:** 1.0

**描述:** 将指定应用激活到前台，让用户能看到

## 功能

- 激活指定应用到前台窗口
- 支持任何Mac应用程序
- 让HCI交互结果可见

## 使用方法

```bash
python3 mac_activate.py "应用名称"

# 示例
python3 mac_activate.py "Finder"
python3 mac_activate.py "Terminal"
python3 mac_activate.py "Safari"
```

---

**平台:** macOS
**依赖:** 系统原生osascript命令
