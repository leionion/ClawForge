# MetaSkillBase Index
Global index of all meta skills, contributors, and usage statistics.


## Structure
- Category: 00-Unclassified ~ 05-Interaction
- Skill Name: Unique lowercase hyphenated name
- Demand Contributor (DC): GitHub ID of demand submitter
- Skill Implementer (SI): GitHub ID of skill developer
- Usage Count: Number of times the skill is used
