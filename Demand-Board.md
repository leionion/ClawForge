# Demand Board
Rules and workflow for submitting, claiming, and implementing human demands.


## 1. Demand Submission (DC Role)
- Submit a clear, actionable human demand.
- Title format: Demand: [Skill Name] – [Function Description]
- Label: demand
- You will be credited as DC for this demand.


## 2. Skill Implementation (SI Role)
- Claim an unclaimed demand.
- Develop an atomic meta skill following core rules.
- Place in the correct category.
- You will be credited as SI for this skill.


## 3. Workflow
Submit Demand → Claim → Develop → Merge → Track Usage Count
