# MetaSkillBase-Core Glossary



## Core Terms



- **Meta Skill**: Atomic, reusable, composable skill unit

- **Bot Skill**: Complex skill composed of multiple Meta Skills

- **Cutter Engine**: Core tool that searches communities and decomposes demands into skills

- **Contributor**: Anyone who submits skills, demands, code, or documentation

- **Usage Count**: Usage Count is a community metric reflecting skill value, recorded and maintained by contributors

- **Elastic Classification**: Classification as inspiration, not restriction—no rejection, only organization

- **OpenClaw Compatible**: All skills work with OpenClaw

# MetaSkillBase-Core 术语表



## 核心术语



- **元技能(Meta Skill)**：原子化、可复用、可组合的技能单元

- **机器人技能(Bot Skill)**：由多个Meta Skills组成的复杂技能

- **Cutter引擎**：在社区间搜索并将需求分解为技能的核心工具

- **贡献者(Contributor)**：提交技能、需求、代码或文档的任何人

- **使用次数(Usage Count)**：衡量技能使用频率的核心指标

- **弹性分类(Elastic Classification)**：分类即灵感，非限制——无拒绝，仅组织

- **OpenClaw兼容(OpenClaw Compatible)**：所有技能均可与OpenClaw配合使用
